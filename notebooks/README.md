# Notebooks

Run in order — each notebook reads outputs written by the previous one.

1. `task1_eda.ipynb` — data extraction, cleaning, EDA, stationarity testing, risk metrics.
2. `task2_modeling.ipynb` — ARIMA/SARIMA and LSTM forecasting models, metric comparison.
3. `task3_forecast.ipynb` — 6-12 month future forecast with confidence intervals.
4. `task4_portfolio_optimization.ipynb` — efficient frontier and portfolio recommendation.
5. `task5_backtesting.ipynb` — strategy vs. benchmark backtest.

See the top-level `README.md` for setup and execution instructions.
